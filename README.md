# DoraSpace
本项目旨在构建能满足日常使用的工具集，目前功能包括：卡片笔记、任务计时、快速链接、日记以及额外的数据看板。
线上地址：https://space.dorakika.cn/
账号密码为坚果云的WebDav账号密码
